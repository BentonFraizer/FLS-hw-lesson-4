§ Конспект урока №4

1. Свойства шрифта

1.1 Свойство font-size

Описание: Свойсво отвечает за размеры шрифта.

Например:

Данный шрифт размером 14px
Данный шрифт размером 20px
Данный шрифт размером 26px

1.2 Свойство font-family

Описание: Свойсво отвечает за то, к какому семейству принадлежит шрифт.

Например:

Этот текст написан шрифтом Roboto. Данный шрифт подключен на страничку при помощи ссылки с сайта Google fonts.
А этот текст написан шрифтом "Lava Pro", которого нет на сайта Google fonts. Шрифт был скачан отдельно, конвертирован соответсвующим образом, скопирован в папку с проектом и подключен на страничку из файла стилей при помощи директивы @font-face.

Типы шрифтов:

  1. С засечками (serif) Пример: "Times New Roman"
  2. Без засечек (sans-serif) Пример: Roboto
  3. Рукописные (script) Пример: "Bad script"
  4. Экранные шрифты (display) Пример: "Press Start 2P"
  5. Моноширинные шрифты (monospace) Пример: "PT Mono"

1.3 Свойство font-style

Описание: Свойсво определяет начертание шрифта — обычное, курсивное или наклонное.

Свойсво font-style имеет следующие значения:

  • normal - обычное начертание текста.
  • italic - курсивное начертание. Содержится в отдельном файле шрифтов.
  • oblique - наклонное начертание. Курсив и наклонный шрифт при всей их похожести не одно и то же. Курсив это специальный шрифт имитирующий рукописный, наклонный же образуется путем наклона обычных знаков вправо.
  • inherit - наследует значение родителя.

1.4 Свойство font-weight

Описание: Свойсво устанавливает насыщенность шрифта. Значение устанавливается от 100 до 900 с шагом 100. Сверхтонкое начертание, которое может отобразить браузер, имеет значение 100, а сверхжирное — 900. Нормальное начертание шрифта (которое установлено по умолчанию) эквивалентно 400 (normal), стандартный полужирный текст — значению 700 (bold).

Пример будет показан на шрифте Monserrat, который имеет все степени толщины от самого тонкого, до самого толстого.

  • 100: Thin;
  • 200: Extra Light (Ultra Light);
  • 300: Light;
  • 400: Normal;
  • 500: Medium;
  • 600: Semi Bold (Demi Bold);
  • 700: Bold;
  • 800: Extra Bold (Ultra Bold);
  • 900: Black (Heavy).

2. Свойства текста

2.1 Свойство color

Описание: Свойсво задаёт цвет шрифта.

Свойство color можно задавать несколькими способами:

  1. orange, blue - цвет задаётся английскими словами, обозначающими соответствующие цвета. CSS предлагает 145 названий шрифтов.
  2. #db4e44 - запись в шестнадцатеричном виде. В первых двух буквах/цифрах "зашифровано" значение для красного канала, в последующих двух - для зелёного, в последних двух - для синего. То есть если вы попробуете задать цвет при помощи следующих значений #FF0000, #00FF00, #0000FF то вы получите красный, зелёный или синий цвета соответственно. Также если у цвета парная запись букв (либо цифр), например #FF9900, то существует сокращенная запись: #F90.
  3. rgb (255, 0, 0) - здесь те же самые 3 (rgb - red, green, blue) канала, как и в шестнадцатеричной записи. Минимальное значение для канала это 0, максимальное - это 255. Таким образом значение rgb (255, 0, 0) задаст красный цвет.
  4. rgba (255, 0, 0, 0.5) - запись аналогична предыдущей за исключением одного момента. Здесь появилось ещё одно значение-так называемый альфа-канал, который отвечает за прозрачность. Значение может быть от произвольным в пределах от 0 до единицы. Если будет 0, то текст будет невидим, т.е. будет полностью прозрачный. Если 1 - то будет отображен в полном цвете, без прозрачности.

2.2 Свойство text-align

Описание: Свойство выравнивает текст.

Свойсво text-align имеет следующие значения:

  1. left. Текст в таком случае будет выравниваться по левому краю.
  2. right. Текст будет выравниваться по правому краю.
  3. center. Текст будет выравниваться по центру.
  4. justify. Текст будет выравниваться по ширине страницы или блока-родителя.
